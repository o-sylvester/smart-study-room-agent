# Smart Study Room Agent

**Lab 1: Building an Agentic AI**  
**Course:** AI PRACTICAL LAB 
**Institution:** Veritas University

---

## Project Overview
This project implements 
an autonomous ** Smart Study Room Agent** 
using the **Perception-Action-Reasoning (PAR) loop** paradigm. The agent continuously monitors environmental conditions (temperature, lighting, noise, air quality, humidity) and autonomously takes actions to optimize the study environment
for maximum focus and comfort.
